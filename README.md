# FindDupes
Win32 command line tool to find duplicate files.


PowerShell code

```PowerShell
echo "Hello World $env:USERNAME"
```


Batch

```Batch
:: comment 1
REM comment 2
echo "Hello World %USERNAME%"
for /f "usebackq tokens=*" %i in (`dir /b *`) do @echo %i

goto :eof
```
